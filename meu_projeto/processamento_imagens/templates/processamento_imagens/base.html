A intenção do processo de classificação é categorizar todos os pixels em uma imagem digital em uma das várias classes de cobertura da terra, ou "temas". Esses dados categorizados podem então ser usados para produzir mapas temáticos da cobertura do solo presente em uma imagem. Normalmente, dados multiespectrais são usados para realizar a classificação e, de fato, o padrão espectral presente nos dados para cada pixel é usado como base numérica para a categorização (Lillesand e Kiefer, 1994). O objetivo da classificação de imagens é identificar e retratar, como um único nível de cinza (ou cor), as características que ocorrem em uma imagem em termos do objeto ou tipo de cobertura do solo que essas características realmente representam no solo.

Com a classificação supervisionada, identificamos exemplos das classes de informação (ou seja, tipo de cobertura do solo) de interesse na imagem. Estes são chamados de "sites de treinamento". O sistema de software de processamento de imagem é então usado para desenvolver uma caracterização estatística da refletância para cada classe de informação. Este estágio é frequentemente chamado de "análise de assinatura" e pode envolver o desenvolvimento de uma caracterização tão simples quanto a média ou a intensidade da refletância em cada faixa, ou tão complexa quanto análises detalhadas da média, variâncias e covariância em todas as faixas. Uma vez que uma caracterização estatística tenha sido alcançada para cada classe de informação, a imagem é então classificada examinando a refletância de cada pixel e tomando uma decisão sobre qual das assinaturas ele se parece mais.
